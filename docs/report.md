# Отчет по практике

## Тема

Генерация DXF-чертежей для лазерной резки.

## Цель работы

Разработать приложение с графическим интерфейсом, которое строит DXF-чертеж деталей корпуса ElectronicsBox для лазерной резки.

## Используемые средства

- Python 3;
- `tkinter` для графического интерфейса;
- `ezdxf` для создания DXF-файла.

## Структура проекта

- `laser_box_generator.py` - запуск программы, окно, ввод размеров и сохранение файла;
- `box_geometry.py` - получение SVG ElectronicsBox и перевод контуров в линии;
- `dxf_writer.py` - запись DXF;
- `svg_writer.py` - запись SVG;
- `assets/ElectronicsBox.svg` - образец внешнего вида;
- `output/dxf/electronics_box.dxf` - готовый файл DXF;
- `tests/` - проверки.

## Алгоритм работы

1. Пользователь вводит длину, глубину, высоту, толщину материала, ширину шипа и зазор.
2. Программа проверяет, что размеры не слишком маленькие.
3. Программа получает SVG по шаблону ElectronicsBox.
4. Размер прямоугольного паза задается через зазор шип-паз.
5. Программа переводит линии SVG в линии DXF.
6. Программа проверяет, что раскладка помещается на лист.
7. Контуры деталей записываются в слой `CUT`.
8. Отверстия и пазы записываются в слой `HOLES`.
9. Результат сохраняется в DXF или SVG.

## Результат

Программа формирует DXF-чертеж деталей коробки ElectronicsBox. Форма раскладки сделана по образцу `ElectronicsBox.svg`: отдельная крышка, дно с пазами, четыре стенки с правильными зубчатыми соединениями и монтажные уголки.

В DXF нет текста, есть только контуры деталей, отверстия и пазы. Единицы DXF - миллиметры.

Для стандартных настроек шип равен `6 x 3 мм`, паз равен `6.1 x 3.1 мм`. Это дает зазор `0.1 мм` по ширине и высоте.

Габарит раскладки для размера `100 x 100 x 100 мм` и материала `3 мм`: `201.5 x 431 мм`.

## Проверка

Проект проверен командами:

```bash
python3 -m compileall -q laser_box_generator.py box_geometry.py dxf_writer.py svg_writer.py tests
python3 -m unittest discover -s tests
python3 laser_box_generator.py --check
python3 laser_box_generator.py --examples
```

Также созданный DXF был открыт обратно через `ezdxf`, проверено отсутствие текста и наличие слоев `CUT` и `HOLES`.

## Вывод

В ходе работы получено GUI-приложение, которое создает DXF-файл деталей ElectronicsBox. Программа позволяет менять размеры, задавать зазор шип-паз и не сохраняет чертеж с явно неправильными параметрами.
