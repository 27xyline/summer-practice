# Отчет по практике

## Тема

Генерация DXF-чертежей для лазерной резки.

## Цель работы

Разработать приложение с графическим интерфейсом, которое строит DXF-чертеж деталей корпуса ElectronicsBox для лазерной резки.

## Используемые средства

- Python 3;
- `tkinter` для графического интерфейса;
- `ezdxf` для создания DXF-файла;
- `xml.etree.ElementTree` для чтения SVG.

## Структура проекта

- `laser_box_generator.py` - запуск программы, окно, предпросмотр и сохранение DXF;
- `box_geometry.py` - чтение SVG и преобразование линий в геометрию программы;
- `dxf_writer.py` - запись линий и текста в DXF;
- `assets/ElectronicsBox.svg` - исходный образец;
- `output/dxf/electronics_box.dxf` - готовый файл DXF;
- `tests/` - проверки.

## Алгоритм работы

1. Программа читает файл `assets/ElectronicsBox.svg`.
2. Из SVG берутся контуры деталей, отверстия и подписи.
3. Координаты переводятся из SVG в обычную систему координат DXF.
4. Кривые линии разбиваются на короткие отрезки.
5. Черные линии записываются в слой `CUT`.
6. Синие линии записываются в слой `HOLES`.
7. Красные подписи записываются в слой `TEXT`.
8. Результат сохраняется в DXF-файл.

## Результат

Программа формирует DXF-чертеж, который повторяет образец `ElectronicsBox.svg`. В чертеже есть верхняя крышка, дно, четыре стенки, монтажные треугольные детали, отверстия, пазы и подписи.

Габарит раскладки по проверке программы: `201.9 x 432.2 мм`. Единицы DXF - миллиметры.

## Проверка

Проект проверен командами:

```bash
python3 -m compileall -q laser_box_generator.py box_geometry.py dxf_writer.py tests
python3 -m unittest discover -s tests
python3 laser_box_generator.py --check
python3 laser_box_generator.py --examples
```

Также созданный DXF был открыт обратно через `ezdxf` и отрендерен в изображение для визуальной проверки.

## Вывод

В ходе работы получено GUI-приложение, которое создает DXF-файл по заданному образцу ElectronicsBox. Чертеж подходит для дальнейшей проверки в программе лазерного станка.
