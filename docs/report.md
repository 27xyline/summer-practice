# Отчет по практике

## Тема

Генерация DXF-чертежей для лазерной резки.

## Выбранное задание

В качестве варианта реализован генератор монтажного короба. Этот вариант соответствует списку заданий из документа практики и подходит для параметризации: пользователь может менять размеры изделия, толщину фанеры и технологические параметры лазерной резки.

## Цель работы

Разработать приложение с графическим интерфейсом для автоматического построения DXF-чертежа деталей монтажного короба из фанеры.

## Используемые средства

- язык программирования Python 3;
- библиотека `tkinter` для GUI;
- собственная запись DXF-файла в текстовом формате без внешних зависимостей.

## Структура проекта

Проект сделан в простой учебной структуре:

- `laser_box_generator.py` - главный файл с GUI и запуском программы;
- `box_geometry.py` - расчёт деталей короба, шипов, пазов и отверстий;
- `dxf_writer.py` - запись рассчитанных линий в DXF-файл;
- `tests/` - простые автоматические проверки.

Такой вариант не смешивает всю программу в одном файле, но остается понятным для уровня первого курса.

## Функции программы

Программа позволяет задать:

- внутреннюю длину короба;
- внутреннюю ширину короба;
- высоту стенок;
- толщину фанеры;
- компенсацию ширины лазерного реза;
- ширину шипов;
- диаметр монтажных отверстий;
- отступ отверстий от края;
- радиус скругления углов;
- расстояние между деталями на листе;
- наличие или отсутствие крышки.

После ввода параметров программа строит предпросмотр и сохраняет DXF-файл.

## Алгоритм генерации

1. Пользователь вводит геометрические параметры изделия.
2. Программа проверяет корректность размеров.
3. По толщине фанеры вычисляются внешние размеры короба и ширина пазов.
4. Для дна строится контур, монтажные отверстия и прямоугольные пазы под стенки.
5. Для передней и задней стенок строятся нижние шипы и вертикальные пазы под боковые стенки.
6. Для боковых стенок строятся нижние шипы и боковые шипы.
7. Для крышки строится отдельная деталь с отверстиями.
8. Все детали размещаются на плоскости с технологическим зазором.
9. Результат записывается в DXF со слоями `CUT` и `INFO`.

## Особенности чертежа

Контуры реза вынесены в слой `CUT`. Информационные подписи вынесены в слой `INFO`. Единицы измерения - миллиметры. В расчетах учитывается компенсация реза, чтобы шипы входили в пазы после лазерной обработки. Нижние пазы дна и нижние шипы стенок строятся из одного набора интервалов. Боковые шипы боковин и вертикальные пазы передней/задней стенки также строятся из общего набора интервалов.

## Примеры результата

В проекте подготовлены два файла:

- `output/dxf/montazhny_korob_3mm.dxf` - вариант для фанеры 3 мм;
- `output/dxf/montazhny_korob_4mm.dxf` - вариант для фанеры 4 мм.

## Проверка

Программа проверена командой:

```bash
python3 -m compileall -q laser_box_generator.py box_geometry.py dxf_writer.py tests
python3 laser_box_generator.py --check
python3 laser_box_generator.py --examples
python3 -m unittest discover -s tests
```

В результате была проверена согласованность соединений, выполнены автоматические тесты и созданы примерные DXF-файлы для фанеры толщиной 3 и 4 мм.

## Вывод

В ходе работы разработано приложение с GUI, которое автоматически формирует DXF-чертеж монтажного короба для лазерной резки. Полученный проект закрывает основные требования практики: есть программа, есть примеры чертежей и есть отчетная документация.
