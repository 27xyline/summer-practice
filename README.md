# Генератор DXF монтажного короба

Проект для практики 1 курса 2025/26 по теме "Генерация DXF чертежей для лазерной резки".

Приложение строит параметрический DXF-чертеж монтажного короба из фанеры. В GUI можно задать размеры, толщину материала, компенсацию реза, ширину шипов, отверстия, скругления и наличие крышки.

## Состав проекта

- `laser_box_generator.py` - главный файл: GUI, запуск, кнопки, командная строка.
- `box_geometry.py` - расчёт размеров, шипов, пазов, отверстий и панелей.
- `dxf_writer.py` - запись результата в DXF-файл.
- `tests/` - простые автоматические проверки.
- `output/dxf/montazhny_korob_3mm.dxf` - пример чертежа для фанеры 3 мм.
- `output/dxf/montazhny_korob_4mm.dxf` - пример чертежа для фанеры 4 мм.
- `docs/report.md` - текст отчета.

## Запуск

```bash
python3 laser_box_generator.py
```

Откроется окно приложения. После изменения параметров нажмите `Обновить предпросмотр`, затем `Сохранить DXF`.

## Генерация примеров без GUI

```bash
python3 laser_box_generator.py --examples
```

Команда создаст DXF-файлы в папке `output/dxf`.

## Проверка соединений

```bash
python3 laser_box_generator.py --check
```

Команда проверяет, что для типовых вариантов 3 и 4 мм шипы и пазы строятся согласованными наборами интервалов.

## Автоматические тесты

```bash
python3 -m unittest discover -s tests
```

Тесты проверяют построение панелей, базовую структуру DXF и согласование соединений.

## Что генерируется

Чертеж содержит:

- дно с пазами под стенки;
- переднюю и заднюю стенки с нижними шипами и вертикальными пазами под боковины;
- левую и правую боковины с нижними шипами и боковыми шипами;
- крышку с отверстиями, если включена соответствующая опция;
- монтажные отверстия;
- слои `CUT` и `INFO`.

Единицы DXF - миллиметры.

## Рекомендации перед резкой

Проверьте DXF в LibreCAD, QCAD, AutoCAD или программе лазерного станка. Для конкретного лазера желательно сделать тестовый маленький паз и уточнить параметр `Компенсация реза`, потому что фактическая ширина реза зависит от станка, скорости и материала.
